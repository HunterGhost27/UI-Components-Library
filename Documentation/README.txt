- The mod is a repository of [i]prebuilt reusable[/i] [b]UI-Components[/b] intended to be used by other mods.

- The mod currently contains:-
  - [b]MessageBox[/b]: An improved version of the [i]message-box[/i]. Has support to display more than 256 characters without crashing the game. Provides the option for larger backgrounds.
    ![url=https://i.imgur.com/Kf0SJLZ.png]msgBox4[/url]
  - [b]ContextMenu[/b]: Allows modification of the [i]context-menu[/i] with custom [b]labels[/b] and [b]actions[/b].
    ![url=https://i.imgur.com/NKOkXNC.png]contextMenu[/url]
  - [b]GMJournal[/b]: A modified GMJournal intended to be used for story-mode journals, in-game manuals and in-game changelogs etc.
    ![url=https://imgur.com/tHcOoGF.png]GMJournal[/url]
